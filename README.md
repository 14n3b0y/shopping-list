# Список покупок — VK Mini App

Простое мини-приложение для ВКонтакте, которое помогает вести список покупок. Добавляйте товары, отмечайте купленное, редактируйте и удаляйте — всё сохраняется в облаке VK и доступно с любого устройства.

🔗 **Работающее приложение:** [vk.com/app54474261](https://vk.com/app54474261)  
📁 **Исходный код:** [github.com/14n3b0y/shopping-list](https://github.com/14n3b0y/shopping-list)

---

## 🚀 Функциональность

- Добавление товара с названием, количеством и примечанием.
- Отметка товара как купленного (чекбокс).
- Редактирование существующего товара.
- Удаление товара.
- Автоматическое сохранение списка в VK Storage.
- Приветствие пользователя по имени и загрузка аватарки.

---

## 🛠 Технологии

- **React** + **TypeScript** — интерфейс и типизация.
- **VKUI** — библиотека компонентов в стиле ВКонтакте.
- **VK Bridge** — взаимодействие с платформой VK (получение данных пользователя, работа с хранилищем).
- **Vite** — сборка проекта.
- **VK Mini Apps Hosting** — деплой приложения.
